#  Student Grade Calculator (Java)

A simple console-based Java project that calculates a student’s total marks, percentage, grade, and pass/fail status.  
This project is designed to strengthen understanding of loops, conditionals, input handling, and basic Java logic building.

---

##  Features

-  Takes student name input  
-  Accepts multiple subjects dynamically  
-  Validates marks (0–100 range)  
-  Calculates total marks & percentage  
-  Assigns grade based on performance  
-  Displays a clean result card  
-  Shows PASS / FAIL status  

---

##  Grading Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100   | A+    |
| 80 - 89    | A     |
| 70 - 79    | B     |
| 60 - 69    | C     |
| 50 - 59    | D     |
| < 50       | F     |

**Pass Mark:** 50% or above
